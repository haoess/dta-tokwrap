NAME

DTA::TokWrap::CxData - DTA tokenizer wrappers: utilities for binary cx-file I/O

DESCRIPTION

DTA::TokWrap::CxData provides some utilities for dealing with *.cx files as written by the dtatw-mkindex(1) program.
